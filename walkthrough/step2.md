## Explore the Island Map

Click the **🗺️ Astro Islands** icon in the activity bar (left sidebar) to open the Island Map panel.

It has four tabs:

**🏝 Islands** — every island in the active file with framework badge, directive, size, and warnings.

**📄 Page** — the assembled page weight: all islands plus their budget against your configured limit.

**🕸 Graph** — an interactive dependency graph. Pages link to islands; islands with shared nanostores stores show dashed state-sharing edges. Click any node to jump to its source file.

**📊 Impact** — the hydration waterfall timeline and the Global User Impact panel, which estimates additional load time by network and region (4G Europe to 2G low-connectivity markets).

> **Tip:** The sidebar updates live as you edit. Save a component file and the size estimates refresh automatically.
